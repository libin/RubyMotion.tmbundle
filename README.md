#RubyMotion.tmbundle

TextMate Bundle for RubyMotion

## Installation

```shell
cd ~/Library/Application\ Support/TextMate/Bundles
git clone git://github.com/libin/RubyMotion.tmbundle.git
```

### TextMate 2.0
```shell
cd ~/Library/Application\ Support/Avian/Bundles
git clone git://github.com/libin/RubyMotion.tmbundle.git
```
